# Revista Belcanto

Este es el repositorio del proyecto "Revista Belcanto", un directorio de negocios locales.

## Cómo empezar

1.  **Instalar dependencias:**
    ```sh
    npm install
    ```

2.  **Iniciar el servidor de desarrollo:**
    ```sh
    npm run dev
    ```

La aplicación estará disponible en `http://localhost:8080`.
